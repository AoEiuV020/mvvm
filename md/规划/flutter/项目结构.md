# Flutter 项目结构

```
lib/
├── main.dart                   # 入口，SDK 初始化
├── app.dart                    # App Widget，路由配置
├── providers/                  # Riverpod Provider 定义
│   ├── sdk_providers.dart      # SDK 服务 Provider
│   └── viewmodel_providers.dart
├── models/                     # 数据模型（来自 SDK）
├── viewmodels/                 # ViewModel 接口和实现
├── views/                      # 页面
├── widgets/                    # 可复用组件
└── router/                     # 路由配置
    └── app_router.dart
```

## 文件命名规范

| 类型 | 命名格式 | 示例 |
|------|----------|------|
| ViewModel | `xxx_viewmodel.dart` | `login_viewmodel.dart` |
| View/Page | `xxx_page.dart` | `login_page.dart` |
| Widget | `xxx_widget.dart` | `message_bubble_widget.dart` |
| Provider | `xxx_providers.dart` | `auth_providers.dart` |
| Model | `xxx_model.dart` | `user_model.dart` |
