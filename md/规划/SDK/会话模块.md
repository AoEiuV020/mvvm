# 会话模块

## 接口定义

```dart
abstract class IConversationService {
  /// 获取会话列表
  Future<List<Conversation>> getConversationList();
  
  /// 获取单个会话
  Future<Conversation> getConversation(String conversationId);
  
  /// 创建私聊会话
  Future<Conversation> createPrivateConversation(String userId);
  
  /// 创建群聊会话
  Future<Conversation> createGroupConversation(List<String> userIds, String groupName);
  
  /// 删除会话
  Future<void> deleteConversation(String conversationId);
  
  /// 置顶会话
  Future<void> pinConversation(String conversationId, bool pinned);
  
  /// 标记已读
  Future<void> markAsRead(String conversationId);
  
  /// 监听会话变化
  Stream<List<Conversation>> get onConversationChanged;
}
```

---

## 数据类型

```dart
class Conversation {
  final String id;
  final ConversationType type;
  final String name;
  final String avatar;
  final Message? lastMessage;
  final int unreadCount;
  final bool pinned;
  final DateTime updatedAt;
}

enum ConversationType { private, group }
```
