# SDK API 文档

本目录包含 IM SDK 的完整接口定义和数据模型。

---

## 目录结构

```
SDK/
├── 概述.md                 # SDK 初始化和模块划分
├── 认证模块.md             # 登录、注册、Token 管理
├── 同步模块.md             # 数据同步
├── 连接模块.md             # TCP 长连接管理
├── 用户模块.md             # 用户信息管理
├── 会话模块.md             # 会话列表管理
├── 消息模块.md             # 消息收发
└── 联系人模块.md           # 好友管理
```

---

## 文档规范

每个模块文档应包含：
- 接口定义（abstract class）
- 数据类型（class/enum）
- 异常类型（Exception）
- 使用示例

**代码规范：**
- 使用 Dart 代码定义接口
- 接口应简洁，只包含签名，不含实现
- 其他框架实现时参照 Dart 接口转换
